# 🎮 小朋友益智遊戲網

一個專為小朋友設計的益智算術遊戲網站，包含兩款精彩遊戲！

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌟 遊戲介紹

### 🌳 果樹成長算術遊戲

培育你的專屬果園！答對算術題目，讓果樹從萌芽慢慢成長到結果！

**遊戲特色：**
- 📚 10以內加法和減法練習
- 🌱 8階段樹木成長動畫
- 🍎 四種果樹：蘋果樹、梨樹、榴蓮樹、芒果樹
- 🎯 打Boss模式：專注培育一棵樹到結果

### 🦸‍♂️ 超級英雄卡片對決

與超級英雄一起戰鬥！答對題目攻擊敵人，保護你的英雄！

**遊戲特色：**
- 🕷️ 4位經典英雄角色
- ⚔️ 即時戰鬥系統
- 💥 答題攻擊敵人
- 🛡️ 答錯會受到傷害

## 📱 支援設備

- 💻 電腦桌面瀏覽器
- 📱 智能手機和平板
- 🔄 響應式設計，自動適應螢幕大小

## 🚀 開始使用

### 方法一：直接訪問（手机版）

前往 GitHub Pages：
```
https://你的用戶名.github.io/你的倉庫名/mobile/
```

### 方法二：本地運行

1. 下載整個 `mobile` 資料夾
2. 使用瀏覽器打開 `index.html`
3. 即可開始遊戲！

## 🎮 如何遊玩

### 果樹成長算術遊戲

1. 選擇答案（兩個選項）
2. 答對 → 樹木成長一階段
3. 答錯 → 再試一次
4. 完成8個階段 → 果樹結果
5. 四棵樹都結果 → 通關！

### 超級英雄卡片對決

1. 聆聽題目或查看提示
2. 選擇正確答案
3. 答對 → 攻擊敵人
4. 答錯 → 受到傷害
5. 敵人HP歸零 → 勝利！
